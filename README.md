## Getting started

```shell 
$ flutter pub add loading_progress_indicator
```

## Usage

``` dart
  body: Center(
    child: LoadingProgressIndicator(
        indicator: BallPulseProgressIndicator(),
        size: 50,
        color: Colors.indigo,
    ),
  ); 
```
